# Informe: Productos y servicios que utilizan Machine Learning

# 1. Duolingo – Personalización del aprendizaje de idiomas

## Nombre del producto o servicio
Duolingo – Plataforma de aprendizaje de idiomas.

## Qué problema resuelve con ML

En una plataforma educativa cada usuario aprende a un ritmo distinto. Algunos estudiantes necesitan repetir ciertos ejercicios mientras que otros pueden avanzar más rápido.

Duolingo utiliza machine learning para **adaptar el contenido a cada usuario**, mostrando ejercicios según su nivel de conocimiento y los errores que comete con más frecuencia.

## Tipo de Machine Learning utilizado

**Aprendizaje supervisado.**

Se considera supervisado porque el sistema aprende a partir de datos donde ya se conoce el resultado, por ejemplo si un usuario respondió correctamente o incorrectamente un ejercicio. Con esa información el modelo puede predecir la probabilidad de que un usuario responda bien una pregunta y decidir qué ejercicio mostrar después.

## Datos que probablemente necesita

- Respuestas correctas e incorrectas de los usuarios  
- Tiempo que tarda el usuario en responder  
- Historial de ejercicios realizados  
- Nivel actual del usuario

# 2. Shazam – Identificación de música

## Nombre del producto o servicio
Shazam – Aplicación de reconocimiento de canciones.

## Qué problema resuelve con ML

Cuando una persona escucha una canción que no conoce, Shazam permite **identificarla grabando unos segundos de audio**.

La aplicación analiza el sonido capturado por el micrófono y lo compara con una base de datos de canciones para encontrar cuál coincide.

## Tipo de Machine Learning utilizado

**Aprendizaje supervisado.**

Se considera supervisado porque el sistema fue entrenado con canciones que ya están identificadas. El modelo aprende los patrones de cada canción y luego puede compararlos con el audio que graba el usuario para encontrar coincidencias.

## Datos que probablemente necesita

- Grabaciones de audio de canciones  
- Características del sonido (frecuencia, ritmo, intensidad)  
- Base de datos de canciones registradas  
- Fragmentos de audio capturados por los usuarios

# 3. Notion AI – Asistencia para escritura

## Nombre del producto o servicio
Notion AI – Asistente de escritura dentro de la aplicación Notion.

## Qué problema resuelve con ML

Muchos usuarios utilizan Notion para escribir notas, documentos o ideas. El asistente de IA ayuda a **generar texto, resumir información o mejorar la redacción** de un documento.

Esto permite ahorrar tiempo cuando se necesita organizar información o redactar contenido.

## Tipo de Machine Learning utilizado

**Aprendizaje supervisado.**

Se considera supervisado porque los modelos de lenguaje se entrenan con grandes conjuntos de textos donde el modelo aprende las relaciones entre palabras y frases. Con esos ejemplos el sistema aprende patrones del lenguaje y puede generar texto coherente o resumir información.

## Datos que probablemente necesita

- Grandes bases de textos para entrenamiento  
- Frases o documentos que escribe el usuario  
- Información que el usuario quiere resumir o reorganizar

# 4. Strava – Análisis de actividades deportivas

## Nombre del producto o servicio
Strava – Aplicación para registrar actividades deportivas.

## Qué problema resuelve con ML

Strava permite registrar actividades como correr o andar en bicicleta usando GPS. La aplicación analiza esos datos para **evaluar el rendimiento del usuario y detectar patrones en sus entrenamientos**.

También puede identificar rutas populares entre los deportistas.

## Tipo de Machine Learning utilizado

**Aprendizaje no supervisado.**

Se considera no supervisado porque el sistema analiza grandes cantidades de actividades registradas por los usuarios para encontrar patrones o agrupaciones, por ejemplo rutas populares o tipos de entrenamiento similares, sin que esos datos estén previamente etiquetados.

## Datos que probablemente necesita

- Datos de GPS de las actividades  
- Distancia recorrida  
- Velocidad promedio  
- Tiempo de actividad  
- Historial de entrenamientos

# 5. Ualá – Detección de fraude en pagos (Argentina)

## Nombre del producto o servicio
Ualá – Sistema de detección de fraude en transacciones.

## Qué problema resuelve con ML

Las aplicaciones financieras deben detectar **compras sospechosas o fraudulentas** para proteger a los usuarios.

Ualá utiliza machine learning para analizar las transacciones y detectar operaciones que no coinciden con el comportamiento normal de una cuenta.

## Tipo de Machine Learning utilizado

**Aprendizaje supervisado.**

Se considera supervisado porque el sistema se entrena con transacciones que ya fueron clasificadas como **fraudulentas o legítimas**. Con esos ejemplos el modelo aprende patrones y puede predecir si una nueva transacción podría ser fraude.

## Datos que probablemente necesita

- Historial de transacciones  
- Ubicación desde donde se realiza la compra  
- Dispositivo utilizado  
- Frecuencia de pagos  
- Comercios donde compra el usuario
