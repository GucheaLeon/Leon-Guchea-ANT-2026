#Analisis de Set de Datos


Dataset: spotify songs 
Link: https://www.kaggle.com/datasets/sujaykapadnis/spotify-songs
Elegí un dataset de canciones de Spotify. Con estos datos se podrían predecir distintas cosas,
por ejemplo si una canción puede llegar a ser popular o no, o incluso estimar aproximadamente su nivel de popularidad. 
Para esto usaría Machine Learning supervisado, porque el dataset ya incluye la variable track_popularity,
entonces el modelo podría aprender la relación entre las características de las canciones y qué tan populares fueron.
Algunas features que usaría serían danceability, energy, tempo y loudness, porque describen cómo suena la canción y si es más movida o intensa,
algo que muchas veces influye en que la gente la escuche más. También usaría acousticness e instrumentalness para saber si la canción es 
más acústica o instrumental, lo cual puede afectar el tipo de público al que le gusta. Otras features útiles serían valence,
que indica si la canción suena más alegre o más triste, y duration_ms, ya que la duración también puede influir en qué tan escuchada es una canción. 
Por último, también podría usar el género de la playlist (playlist_genre o playlist_subgenre),
porque algunos géneros suelen ser más populares que otros dependiendo del público y del momento.
